# 宏的个人 IP 主页改版参考图

## 核心定位

- 人物：宏，15 年一线实战的微信生态全链路运营专家，程序员出身。
- 领域：微信小店、视频号直播、微信豆投放、微信推客、小程序商城、AI 工具与 GEO。
- 价值主张：只分享经实战验证、可落地的解决方案。
- 目标感受：专业、可信、懂技术、懂运营、能落地，不像企业官网，也不像泛知识博主模板。

## 视觉系统

- 概念主线：实战档案馆（Archive / Dossier）。
- 主色：墨黑 `#111318`、纸白 `#F7F7F2`。
- 品牌色：微信绿 `#07C160`。
- 辅助色：钴蓝 `#2F5BFF`、少量朱红 `#FF5A45`。
- 字体气质：中文现代黑体为主，少量宋体用于观点和引语。
- 材质：纸张、索引标签、细网格、标注线、项目照片、规则文档裁切。
- 圆角：0-8px，避免大面积圆角卡片。
- 图片处理：真实商务照和微信公开课现场照，轻微冷色调统一，不改变人物身份。
- 动效想象：滚动揭示、索引线移动、图片轻微视差、知识条目展开、关键词高亮。
- 禁止：紫蓝 AI 渐变、玻璃拟态堆叠、悬浮光球、无意义数据图、卡片套卡片、重复左右分栏。

## 生成约束

- 每张图只表现一个板块，横向 16:9。
- 桌面参考尺寸为 1536 x 1024 级别，内容宽度按 1440px 网站理解。
- 所有图共享同一导航、字体、色彩、线条、图片调色和按钮语言。
- 文字只使用真实资料中的信息，不编造数据、客户评价或品牌背书。
- 二维码在参考图中作为真实资产展示，不重新绘制，不改变码点结构。
