## 一、一起买是什么？有什么用？

#### 定义

「一起买」是微信小店推出的一种社交拼团营销工具。商家创建活动后，用户可邀请好友一起购买，达到指定人数后即可享受拼团折扣价。它本质上是一种“社交裂变+促销”的玩法。

#### 核心作用

**对商家：**

**提升转化率：**拼团折扣形成价格优势，能有效促进犹豫用户下单。

**带来社交裂变：**用户为了成团，会主动将活动分享到微信聊天或朋友圈，为店铺带来免费的社交传播和精准新用户。

**快速冲量：**适合新品破零、爆品打造、节点大促时集中冲销量。

**对带货者（达人）：**

**带货佣金：**带货者可以在直播间发起商家设置的「一起买」活动，引导用户参团，并获得相应的带货佣金。

**提升直播间互动：**主播引导观众“点一下拼团”，能有效提升直播间互动氛围和停留时长。

**影响：**「一起买」是微信小店利用微信社交生态进行裂变的核心工具。与传统的“满减”相比，它多了一层“人拉人”的社交传播属性，能带来成本更低的增量流量。

## 二、一起买有哪些具体规则及条件？

### 1. 商家创建规则

*下表为对应规则整理。*

| 规则项 | 具体说明 |
| --- | --- |
| 活动类型 | 支持单品活动（一个商品拼团）和多品活动（多个商品组合拼团）。 |
| 价格设定 | 商家自行设置拼团折扣价，需低于原价。 |
| 成团规则 | 商家可设置成团人数（如2人、3人、5人团）。用户成功邀请足够人数参团后，活动才生效。 |
| 活动时长 | 商家可设置活动的时间范围。 |
| 商品库存 | 商家需为活动设置独立的库存。 |
| 退款规则 | 用户支付后，若长时间（通常为24小时）未成团，系统会自动取消订单并退款。 |
| 适用场景 | - 商品详情页（单品活动） - 店铺首页（多品活动） - 直播间 - 分享海报/链接 |
| 操作路径 | 微信小店后台 → 营销中心 → 一起买 → 新建活动。 |

### 2. 带货者（达人）参与规则

*下表为对应规则整理。*

| 规则项 | 具体说明 |
| --- | --- |
| 前提条件 | 商家必须在后台创建了一个「一起买」活动，并将该商品添加到了你的橱窗或直播间。 |
| 发起方式 | 在直播间的购物车中，找到带有“一起买”标签的商品，点击发起活动。 |
| 佣金计算 | 用户通过你发起的拼团链接下单，你仍可获得正常的带货佣金。 |
| 流量撬动 | - 直播间观众可以将活动分享到聊天/朋友圈，邀请好友参团。 - 观众点击活动页面右下角的爱心赞，活动有机会被推荐到微信服务号盒子和订单与卡包页面，获得更多免费曝光。 |
| 操作路径 | 直播间 → 购物车 → 找到带有“一起买”标签的商品 → 发起活动。 |

## 三、操作流程详解

#### 商家端操作流程

**开通：**登录微信小店后台 → 营销中心 → 一起买。首次使用需开通功能并同意协议。

**新建活动：**点击【新建活动】，选择【单品活动】或【多品活动】。

**设置活动：**

**活动名称/时间：**设置活动的名称、开始和结束时间。

**选择商品：**从店铺商品中选择要参与拼团的商品。

**设置价格：**填写拼团折扣价（低于原价）。

**成团人数：**设置需要多少人才能成团（如2人、3人）。

**库存：**设置独立的活动库存。

**发布活动：**确认设置无误后，点击【发布】。活动将自动在商品详情页展示“一起买”标签。

#### 带货者（达人）端操作流程

**选品：**在优选联盟选品中心或商家提供的商品列表中，找到带有“一起买”标签的商品，并将其上架到橱窗或加入直播间购物车。

**发起拼团：**开播后，在直播间购物车中，找到该商品，点击并选择【发起一起买】。

**引导用户：**主播在直播间话术引导：“家人们！这个商品现在有拼团活动，点一下购物车，再点这个‘一起买’按钮，就能邀请你朋友一起来买，更划算！”

**用户操作：**观众点击后，可立即参团，或分享链接到微信邀请好友。

## 四、具体案例与应用场景

**案例1：**新店开业，利用拼团实现“裂变破零”

**场景：**王老板开了一家水果店，上架了“5斤装海南贵妃芒”。他想快速积累首批销量和评价，但传统销售模式很难实现。

**操作：**

王老板在后台创建一个单品活动，设置原价68元，拼团价48元，2人成团。

**在直播间和社群宣传：**“新店开业！贵妃芒原价68，今天两人成团只要48！快拉上你的闺蜜一起买！”

**结果：**用户为了享受低价，主动把链接分享给朋友，形成了社交裂变。活动期间，该商品销量超过500单，其中50%的订单来自用户分享带来的新客户。

**案例2：**带货达人直播间冲量

**场景：**一名带货达人小C，在618大促期间，要为一家合作的服装店带货。

**操作：**

**商家配合：**该服装店提前创建了一个多品活动，将“夏季连衣裙”和“防晒外套”组成套餐，原价399元，拼团价299元，2人成团。

**达人发起：**小C在直播时，将这款套餐上架到购物车，并点击发起“一起买”。

**引导话术：**“姐妹们！这个夏季清凉套餐真的好划算！原价399，现在你只要拉上你的闺蜜，两个人一起买，只要299！省了100块！”

**观众操作：**观众点击后，两人成团即可下单。

**结果：**小C的直播间氛围热烈，用户积极分享，最终该套餐在直播间的销量是普通商品的2.5倍。

**案例3：**利用“爱心赞”免费获取公域流量

**场景：**某母婴品牌在“六一儿童节”期间，设置了一场“一起买”活动。

**操作：**

**品牌方设置好活动后，在直播间引导用户：**“大家点一下右下角的爱心赞，这个活动就有机会被推荐给更多人看到！”

用户点赞后，该活动在一定概率上，出现在了微信服务号盒子和订单与卡包页面的推荐位中。

**结果：**该活动通过“爱心赞”触达了大量非直播间粉丝的潜在用户，获得了可观的免费公共推荐流量，为品牌带来了破圈曝光。

## 五、商家/达人日常如何运用好这个能力？

### 1. 商家日常运营与优化清单

*下表为对应规则整理。*

| 场景 | 具体操作 | 目的 |
| --- | --- | --- |
| 新品破零 | 为新品设置一个有吸引力的2人成团价。 | 利用老用户的朋友圈，低成本获取首批订单和评价。 |
| 节点大促 | 在618、双11等节点，将主推款和凑单品组合成多品活动，设置3人成团。 | 提升客单价，同时通过社交裂变获取新客。 |
| 日常促销 | 选择库存充足、消费者经常需要下单的商品（如纸巾、零食），设置长期有效的2人成团活动。 | 日常维持销量，持续获取低价裂变流量。 |
| 清理库存 | 对需要清仓的商品，设置超低拼团价（如原价80，拼团价50），并配合直播推广。 | 快速回笼资金，同时利用用户分享减少清仓亏损。 |
| 数据分析 | 通过后台【营销中心 - 一起买】的数据看板，查看活动的参与人数、成团率、新增访客数。 | 评估活动效果，优化下次活动的价格和成团人数设置。 |

### 2. 带货者（达人）如何运用

**主动选用：**在优选联盟选品时，优先选择带有“一起买”标签的商品，因为它们自带裂变属性，更容易出单。

**直播间话术优化：**直播时，不要只介绍商品，要明确告知观众：“点一下购物车，再点一下‘一起买’，邀请你的朋友一起来，两个人下单更便宜！”

**引导用户点赞：**直播中，主播可以多引导观众，点击活动页面的“爱心赞”，因为点赞能帮你撬动微信官方的免费公域推荐流量。

## 六、注意事项

**合规宣传：**严禁“好评返现” 或通过优惠券变相诱导用户进行好评。同时，在宣传优惠券时，要明确使用门槛和限制条件，避免虚假宣传。

**优惠券冲突：**「店铺优惠券」和「一起买」价格优惠不能同时生效，系统优先自动触发「一起买」。因此，商家在设置活动时，要考虑到用户同时拥有优惠券和拼团机会时的体验。

**成团率是关键：**成团人数设置需合理。2人团是最容易成功的，5人团失败率很高。建议新手商家从2人团开始，成功后逐步尝试3人团。

**关注售后：**用户发起拼团但最终失败，系统会自动退款。商家需要处理好这些退款订单，避免因此导致店铺体验分下降。同时，对于拼团成功后的订单，及时发货，做好服务。

**注意“价保”规则：**如果你同时为商品设置了【价格保护】，而拼团价低于日常售价，用户可能会发起价保申请（即使他没参与拼团）。因此，不要给设置了价保的商品，同时设置一个极低折扣的拼团价，以免产生意外成本。

**总结：**「一起买」是微信小店生态中“社交裂变”的核心武器。商家应善用“单品活动”破零，利用“多品活动”提客单，结合直播间“爱心赞”获取免费公域流量。核心原则是：价格有吸引力，成团门槛低（2-3人优先），社交裂变是核心。
