## 一、小店会员是什么？有什么用？

#### 定义

小店会员是微信小店为商家提供的店内原生会员解决方案。它无需开发，商家开通后即可快速创建并管理自己的会员体系，包括会员等级、权益、积分规则等。用户可在店铺首页、商品详情页、订单页、直播间购物袋等多个场景一键成为会员，享受专属权益。

#### 核心价值

**对商家：**

**构建私域用户池：**将公域流量（直播间、商品页）的访客，转化为可长期运营的店铺会员，建立自己的用户资产。

**提升复购与忠诚度：**通过会员等级权益（如专享券、折扣）、积分体系（积分兑换商品）来激励用户持续复购，提升用户生命周期价值。

**简化运营操作：**无需开发，后台即可配置，门槛低，适合所有类型商家快速上手。

**对用户：**

**享受专属权益：**成为会员后，可享受开卡礼、会员专享券、积分兑换、经验值升级等专属服务。

**提升购物体验：**会员身份在店内有明显标识，购物时可自动享受优惠，体验感更好。

**影响：**小店会员功能是微信小店打通“引流→转化→复购→裂变”全链路私域运营的核心基础设施。

## 二、小店会员有哪些具体规则及条件？

### 1. 商家开通条件

*下表为对应规则整理。*

| 条件 | 具体要求 |
| --- | --- |
| 店铺状态 | 微信小店处于正常经营状态，且满足平台基础运营要求。 |
| 管理员确认 | 开通过程需要小店超级管理员微信扫码确认。 |

### 2. 功能配置规则

#### 等级与权益配置

*下表为对应规则整理。*

| 配置项 | 规则说明 |
| --- | --- |
| 经验值规则 | 可按消费金额、消费次数设置会员经验值获取规则。支持成功设置后重新修改，无修改上限。 |
| 会员等级卡片 | 最高可添加至 「会员等级VIP 10」。 【不可逆】：保存后等级不可删除；仅支持修改 「会员名称」，不支持修改「等级经验值要求」与「经验值倍数」。 |
| 开卡礼 | 用户入会时直接下放的会员礼券（最多五张），无需手动领取。配置成功后，可修改发放数量、到期时间。 |
| 会员专享券 | 用户入会后，需在设置时限内手动领取的优惠券。支持设置不同类型、使用方式与发放门槛。配置成功后，可修改发放数量与到期时间。 |

#### 积分配置规则

*下表为对应规则整理。*

| 配置项 | 规则说明 |
| --- | --- |
| 积分规则 | 可按消费金额、消费次数设置积分获取和抵扣规则。 |
| 积分商品 | 积分商品仅在积分商城兑换，不在店铺正常展示。支持按商品SKU维度设置，系统默认按积分比例预填积分。 |

#### 入口与状态管理

*下表为对应规则整理。*

| 操作项 | 规则说明 |
| --- | --- |
| 开启入口 | 完成配置后，需手动点击【开启入口】。开启后在店铺首页、商品详情页、订单详情页、直播间购物袋展示。 |
| 关闭入口 | 可随时关闭，需小店管理员扫码确认。关闭后历史会员关系、等级卡、优惠券与积分等配置依旧保存，可随时重新开启。 |
| 注销功能 | 注销后，历史会员关系、等级卡、优惠券与积分等配置将全部清空，请谨慎操作。 |

### 3. 用户参与规则

*下表为对应规则整理。*

| 场景 | 规则说明 |
| --- | --- |
| 加入会员 | 通过直播间、店铺首页、商品详情页、订单详情页等入口，一键成为会员。 |
| 使用权益 | 从店铺首页进入会员中心，查看等级与积分，使用优惠券或积分兑换商品。 |
| 注销会员 | 用户可手动注销。注销后，在该店铺累积的经验值、积分、优惠券等权益全部清空。重新加入后不再享受原权益。 |

### 4. 与“小程序会员服务”的区别

*下表为对应规则整理。*

| 对比项 | 小店会员（原生） | 小程序会员服务（关联） |
| --- | --- | --- |
| 开发成本 | 无需开发，后台直接配置 | 需开发（自有或找服务商），或购买服务市场工具 |
| 会员体系 | 基于微信小店内置的等级、积分、券体系 | 基于商家自有小程序（如品牌APP的会员体系） |
| 适用场景 | 快速建会，满足基础运营需求 | 品牌商家需要与自有小程序会员数据打通，实现全场景服务 |
| 数据打通 | 数据仅在小店内使用 | 可与小程序的会员数据互通，实现同步等级、积分等 |
| 准入要求 | 满足平台基础运营要求 | 需满足热招品牌准入要求 |

## 三、操作流程详解（商家端）

#### 开通会员功能

**路径：**登录微信小店后台 → 用户运营 → 会员营销 → 【开通】。

**确认：**需小店超级管理员微信扫码确认。

#### 配置会员功能

**路径：**开通后，在「会员营销」页面进行配置。

### 2.1 配置会员等级与优惠券（四步法）

*下表为对应规则整理。*

| 步骤 | 操作 | 说明 |
| --- | --- | --- |
| 第一步 | 设置经验值 | 按消费金额/消费次数设置。规则可修改。 |
| 第二步 | 设置会员等级卡片 | 新建等级（至多VIP10）。等级不可删除；仅可修改名称。 |
| 第三步 | 权益配置-开卡礼 | 创建入会即自动下发的礼券（最多5张）。 |
| 第四步 | 权益配置-会员专享券 | 创建入会后需手动领取的专属券。 |

### 2.2 配置积分规则与商品（两步法）

*下表为对应规则整理。*

| 步骤 | 操作 | 说明 |
| --- | --- | --- |
| 第一步 | 设置积分规则 | 按消费金额/消费次数设置积分获取与抵扣规则。 |
| 第二步 | 设置积分商品 | 创建仅限积分商城兑换的商品。 |

#### 开启会员入口

**操作：**完成配置后，手动点击 【开启入口】。

#### 关闭/注销会员功能

**关闭：**可随时关闭入口，配置保留。

**注销：**需谨慎操作，所有配置与会员数据将清空。

## 四、具体案例与应用场景

**案例1：**新店快速搭建会员体系，提升复购

**场景：**王老板新开了一家零食小店，他想建立会员体系，让老客能持续复购。

**操作：**

开通「小店会员」功能。

**设置经验值规则：**消费1元=1经验值。

**新建两个等级：**普通会员（0-100经验值） 和 银卡会员（100+经验值）。

**配置开卡礼：**入会即送「满50减5」券一张。

**配置会员专享券：**银卡会员每月可领「满100减10」券。

**设置积分规则：**消费1元=1积分，500积分可兑换「店铺通用10元券」。

**结果：**开业一周，店铺增加了500名会员。开卡礼带动了新客首单转化，银卡会员的专属券则激励着用户持续消费升级，店铺复购率提升了20%。

**案例2：**配合直播间，快速拉新入会

**场景：**一家美妆店在618大促期间进行直播，想快速积累会员。

**操作：**

**配置一个非常有吸引力的开卡礼：**「满200减30」专用券。

**直播间主播话术：**“家人们！为了庆祝618，现在加入我们的会员，立刻送你一张满200减30的券！券后更划算！赶紧点击下方小卡片，一键入会！”

用户在直播间购物袋看到“一键入会”入口，点击即成为会员，并自动获得这张大额券。

用户使用该券下单，完成首单转化。

**结果：**这场直播，该店铺新会员转化率比平时提升了35%，会员活动也带动了直播间整体的销量增长。

## 五、商家/达人日常如何运用好这个能力？

### 1. 商家日常运营与优化清单

*下表为对应规则整理。*

| 频率 | 操作 | 目的 |
| --- | --- | --- |
| 每周 | 检查后台会员数据：会员增长数、等级分布、开卡礼使用率、会员专享券核销率。 | 分析会员活跃度和权益吸引力，便于优化策略。 |
| 每月 | 更新会员权益：根据库存和成本，调整开卡礼和会员专享券的设置（如更换为当季热销品）。 | 保持会员权益对用户的持续吸引力。 |
| 每季度 | 复盘会员体系：评估现有等级和积分设置是否合理。如果会员升级速度过慢，可考虑调整经验值倍数或积分规则。 | 确保会员体系能有效激励用户成长和复购。 |
| 大促/新品 | 为重要节点（如618、双11）或新品上市，配置专属的会员优惠券或积分兑换商品。 | 利用会员体系为节点大促和新品预热造势。 |

### 2. 带货者（达人）如何运用

**理解会员价值：**达人应了解所带货商家的会员权益。在直播中，可以对观众说：“家人们记得先去店铺首页加入会员，有开卡礼可以领！”

**配合商家活动：**商家如在大促期间有会员专属活动（如会员双倍积分），达人可在直播中配合宣传，增加用户性价比感知，促进成交。

## 六、注意事项

**不可逆操作：**

会员等级保存后不可删除。配置等级时需合理规划，避免创建后无法撤销。

等级经验值要求与经验值倍数保存后不可修改（仅可修改名称）。等级规则的设计需要深思熟虑。

**积分与等级的关系：**积分与经验值是独立的。经验值决定等级，积分用于兑换商品。建议两者配合使用，相辅相成。

**合规使用：**不得利用会员功能进行非法集资、传销或侵犯用户隐私的行为。所有会员权益的设置需合法合规。

**与“小程序会员”不冲突：**商家可以同时开通“小店会员”和“小程序会员服务”。小店会员满足基础会员运营需求，小程序会员满足深度品牌会员打通需求，两者并行不悖。

**开通API接口：**如需实现更深度的功能对接（如自定义会员卡等），商家可自行申请开通相应的API接口权限，经平台审核通过后方可使用。

**维护体验分：**所有会员运营的前提，是提供优质的商品和服务。低体验分会导致店铺整体失去搜索和推荐流量，再好的会员体系也难以发挥作用。

**总结：**微信小店「小店会员」是商家构建私域用户池、提升复购率的核心利器。其“低门槛、易配置、高产出”的特点，非常适合所有微信小店商家使用。核心是：善用等级激励用户、用好积分促进复购、注重权益的持续吸引力。
